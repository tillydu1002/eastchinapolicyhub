# 华东政务信息站点

腾讯华东公共事务团队月度政务信息站点（对内参考）。

## 快速开始

- **本地预览**：双击根目录下的 `预览.bat`
- **上线部署**：见 `上线部署清单.md`

## 目录结构

```
.
├── index.html              # 仪表盘首页：统计卡片 + 最新 5 条 + 归档入口
├── archive/
│   └── YYYY-MM.html        # 历史月度完整版（5 大模块表格）
├── data/
│   ├── _schema.md          # 数据字段规范
│   └── YYYY-MM.json        # 月度结构化数据源
├── scripts/                # 构建 / 同步脚本
├── .nojekyll               # GitHub Pages 必备（禁用 Jekyll 构建）
├── 预览.bat                # 本地一键启动 http 服务
└── 上线部署清单.md          # 发布到 GitHub Pages 的操作步骤
```

## 内容规范

### 五大模块（固定顺序）

1. 人事变动（副部级及以上）
2. 领导动态
3. 重大政策
4. 行业信息
5. 友商动态

### 区域合并

上海市市级 + 徐汇区合并为"上海"，不做区分。

### 更新节奏

- 每日 09:30 前：补录前一日信息
- 每日 18:00 前：补录当日动态
- 每月 1 日 10:00：上线新月度页，上月页面锁版归档

## 视觉风格

- 主色：深藏青 `#0f2a5f`（权威、稳重）
- 点缀：暖金 `#b8924a`（高级感分隔）
- 底色：暖米 `#fdfbf5`（护眼、纸质感）
- 字体：思源宋体 / PingFang / 微软雅黑 回退
